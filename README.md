# 欢迎使用 Cxcybot
**Cxcybot 是一个解决比赛信息下发不及时的自动推送机器人，使热衷于比赛的同学们不错过每一份比赛信息**

## 使用方法
安装 `python3`

注册server酱

克隆下来此项目

> pip install -r requirements.txt

将`main.py`中server酱ID更换成自己的ID

运行程序
>python main.py
